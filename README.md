# نظام إدارة شؤون الموظفين المتكامل
## Employee Management System

نظام شامل ومتكامل لإدارة شؤون الموظفين مع محرك ذكي لحساب العلاوات والترفيعات الوظيفية.

## 🌟 المميزات الرئيسية

### 📊 لوحة التحكم الرئيسية
- إحصائيات سريعة عن الموظفين
- عرض الموظفين المستحقين للعلاوات
- واجهة عصرية وجذابة باللغة العربية

### 👥 إدارة الموظفين
- إضافة موظفين جدد مع جميع البيانات المطلوبة
- عرض قائمة الموظفين مع إمكانية البحث والفلترة
- ملفات شخصية تفصيلية لكل موظف

### 🧠 المحرك الذكي للحسابات
- حساب تلقائي للعلاوات والترفيعات
- تطبيق قواعد الترفيع المختلفة حسب الدرجة الوظيفية
- مراعاة تأثير الأحداث المهنية على مواعيد الاستحقاق
- حساب دقيق للخدمة الفعلية

### 📋 إدارة الأحداث المهنية
- كتب الشكر والتقدير مع تأثيرها على مدة الاستحقاق
- العقوبات الإدارية وتأثيرها
- الإجازات الطويلة وتجميد الخدمة
- الحصول على شهادات عليا
- أحداث مخصصة

### 📈 التقارير والإحصائيات
- تقارير شاملة عن حالة الموظفين
- إحصائيات العلاوات والترفيعات
- تتبع الأحداث المهنية

## 🚀 كيفية التشغيل

### المتطلبات
- Python 3.8 أو أحدث
- نظام التشغيل: Windows, macOS, أو Linux

### خطوات التثبيت

1. **تحميل المشروع**
```bash
git clone <repository-url>
cd employee-management-system
```

2. **تثبيت المتطلبات**
```bash
pip install -r requirements.txt
```

3. **تشغيل النظام**
```bash
python run.py
```

أو تشغيل الملف الرئيسي مباشرة:
```bash
python main.py
```

## 🎯 كيفية الاستخدام

### 1. إضافة موظف جديد
- انقر على "إضافة موظف جديد" من القائمة الجانبية
- املأ جميع البيانات المطلوبة
- انقر على "حفظ الموظف"

### 2. عرض قائمة الموظفين
- انقر على "قائمة الموظفين"
- استخدم حقل البحث للعثور على موظف محدد
- انقر نقراً مزدوجاً على اسم الموظف لفتح ملفه

### 3. إضافة حدث مهني
- انقر على "إضافة حدث مهني"
- حدد الموظفين المطلوبين
- اختر نوع الحدث
- املأ التفاصيل المطلوبة
- انقر على "حفظ الحدث"

### 4. عرض ملف الموظف
- يعرض الملف معلومات شاملة عن الموظف
- المؤشرات الرئيسية (الدرجة الحالية، الاستحقاق القادم، الخدمة الفعلية)
- تبويبات للمسار الوظيفي والأحداث المهنية

## ⚙️ قواعد النظام

### قواعد الترفيع
- **الدرجات 10-6**: 3 علاوات + 1 ترفيع (نظام 4/X)
- **الدرجات 5-2**: 4 علاوات + 1 ترفيع (نظام 5/X)
- **الدرجة 1**: 10 علاوات فقط (لا يوجد ترفيع)

### تأثير الأحداث المهنية
- **كتب الشكر**: تقليص 1-6 أشهر من مدة الاستحقاق
- **الشهادات العليا**: تقليص 12 شهر
- **العقوبات**: إضافة 3-12 شهر حسب نوع العقوبة
- **الإجازات الطويلة**: تجميد احتساب الخدمة

## 🔧 الميزات التقنية

- **واجهة المستخدم**: CustomTkinter لواجهة عصرية
- **قاعدة البيانات**: SQLite محلية
- **المحرك الذكي**: خوارزميات متقدمة لحساب الاستحقاقات
- **دعم اللغة العربية**: واجهة كاملة باللغة العربية
- **التوافق**: يعمل على جميع أنظمة التشغيل
