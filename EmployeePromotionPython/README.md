# نظام أتمتة العلاوات والترفيعات الوظيفية

## Employee Promotion Automation System

نظام متكامل وشامل لأتمتة وحوسبة العمليات المعقدة المتعلقة بحساب العلاوات السنوية والترفيعات الوظيفية لجميع الموظفين من الكادر التدريسي والإداري.

---

## 🌟 المميزات الرئيسية

### ✅ إدارة شاملة للموظفين
- إضافة وتعديل وحذف الموظفين
- ملف تفاعلي شامل لكل موظف
- تتبع المسار الوظيفي الكامل
- حساب الخدمة الفعلية تلقائياً

### ✅ المحرك الذكي للحسابات
- حساب العلاوات والترفيعات تلقائياً
- تطبيق قواعد مؤشر التتبع المعقدة
- إعادة الحساب عند إضافة أحداث قديمة
- دعم جميع الدرجات الوظيفية (1-10)

### ✅ نظام الأحداث المهنية الشامل
- كتب الشكر والتقدير مع القيود التلقائية
- العقوبات الإدارية (لفت نظر، إنذار، توبيخ)
- الإجازات الطويلة (بدون راتب، رعاية معاقين، خمس سنوات، أمومة)
- الشهادات الأعلى مع تحديث البيانات
- الأحداث المخصصة

### ✅ التقارير والإحصائيات المتقدمة
- لوحة تحكم تفاعلية
- رسوم بيانية متقدمة
- تقارير مفصلة للموظفين والأحداث
- الاستحقاقات القادمة
- تصدير البيانات (CSV, Excel, PDF)

### ✅ واجهات عصرية ومتجاوبة
- تصميم عصري مع Bootstrap 5
- دعم كامل للغة العربية (RTL)
- واجهات تفاعلية مع JavaScript
- رسوم بيانية مع Chart.js
- تأثيرات بصرية جذابة

## 📦 التثبيت والتشغيل

### الطريقة الأولى: التثبيت السريع

```bash
# 1. تحميل المشروع
git clone https://github.com/your-repo/EmployeePromotionPython.git
cd EmployeePromotionPython

# 2. إنشاء بيئة افتراضية (مستحسن)
python -m venv venv

# 3. تفعيل البيئة الافتراضية
# على Windows:
venv\Scripts\activate
# على Linux/Mac:
source venv/bin/activate

# 4. تثبيت المتطلبات
pip install -r requirements.txt

# 5. تشغيل التطبيق
python run.py
```

### الطريقة الثانية: التثبيت اليدوي

```bash
# 1. إنشاء مجلد المشروع
mkdir EmployeePromotionSystem
cd EmployeePromotionSystem

# 2. تثبيت المكتبات
pip install Flask Flask-SQLAlchemy Flask-Migrate python-dateutil

# 3. نسخ ملفات المشروع
# (نسخ جميع الملفات من المستودع)

# 4. تشغيل التطبيق
python run.py
```

## 🔧 الإعدادات والتخصيص

### إعدادات قاعدة البيانات
يمكن تغيير إعدادات قاعدة البيانات في ملف `config.py`:

```python
# SQLite (افتراضي)
SQLALCHEMY_DATABASE_URI = 'sqlite:///employee_promotion.db'

# MySQL
SQLALCHEMY_DATABASE_URI = 'mysql://user:password@localhost/database_name'

# PostgreSQL
SQLALCHEMY_DATABASE_URI = 'postgresql://user:password@localhost/database_name'
```

### إعدادات الأمان
```python
# تغيير المفتاح السري (مهم للإنتاج)
SECRET_KEY = 'your-secret-key-here'

# تفعيل/إلغاء وضع التطوير
DEBUG = True  # False للإنتاج
```

### إعدادات الخادم
```python
# في ملف run.py
app.run(
    host='0.0.0.0',  # للوصول من أجهزة أخرى
    port=5000,       # رقم المنفذ
    debug=True       # وضع التطوير
)
```

## 📚 دليل الاستخدام

### 1. إضافة موظف جديد
1. اذهب إلى **الموظفين** > **إضافة موظف جديد**
2. املأ جميع البيانات المطلوبة:
   - الاسم الرباعي واللقب
   - تاريخ المباشرة
   - تاريخ آخر استحقاق للعلاوة
   - آخر تاريخ استحقاق للترفيع
   - مؤشر تتبع العلاوة
   - الشهادة العلمية
   - صنف الوظيفة
   - العنوان الوظيفي
   - الدرجة الوظيفية
3. اضغط **حفظ الموظف**

### 2. إضافة حدث مهني
1. اذهب إلى **الأحداث المهنية** > **إضافة حدث مهني**
2. حدد الموظفين المطلوبين
3. اختر نوع الحدث
4. املأ التفاصيل حسب نوع الحدث
5. اضغط **حفظ الحدث**

### 3. عرض ملف الموظف
1. اذهب إلى **الموظفين**
2. اضغط على **عرض** بجانب اسم الموظف
3. ستظهر لك:
   - المؤشرات الرئيسية
   - المسار الوظيفي
   - الأحداث المهنية
   - الاستحقاق القادم

### 4. عرض التقارير
1. اذهب إلى **التقارير**
2. اختر نوع التقرير المطلوب
3. طبق الفلاتر المناسبة
4. اضغط **تصدير** لحفظ التقرير

## 🔒 الأمان والحماية

### إعدادات الأمان المطبقة
- **CSRF Protection**: حماية من هجمات Cross-Site Request Forgery
- **SQL Injection Prevention**: استخدام SQLAlchemy ORM
- **XSS Protection**: تنظيف المدخلات تلقائياً
- **Session Security**: جلسات آمنة مع انتهاء صلاحية

### توصيات للإنتاج
1. **تغيير المفتاح السري**:
   ```python
   SECRET_KEY = 'your-very-secure-secret-key'
   ```

2. **إلغاء وضع التطوير**:
   ```python
   DEBUG = False
   ```

3. **استخدام HTTPS**:
   ```python
   app.run(ssl_context='adhoc')
   ```

4. **تحديد عنوان IP محدد**:
   ```python
   app.run(host='127.0.0.1')  # بدلاً من 0.0.0.0
   ```
